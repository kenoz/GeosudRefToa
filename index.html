Geosud : Reflectance TOA

1. What is Equipex Geosud ?

The Equipex GEOSUD ("GeoInformation for Sustainable Development") is an initiative coordinated by Irstea (UMR TETIS) to serve the scientific and management public actors (state agencies, local authorities, associations, ...) funded in the framework of the French Programme « Investissements d’Avenir » . Equipex GEOSUD aims at establishing a national satellite data infrastructure freely accessible to public actors. It will :
  • provide for five years (2011-2015) annual high resolution satellite coverage of France and of regions in South countries,
  • make processing capacities and tools available to public actors, 
  • develop innovative methods for data processing and analysis,
  • support the networking of scientific community and the community of land management actors.
For more informations : see the official website equipex-geosud.fr

2. Plugin functionnalities

The plugin is dedicated to Geosud data. However, it should be used for satellite images acquired in other ways. It converts Digital Numbers (DN) to Top of Atmosphere (ToA) Reflectance for three types of instrument : RapidEye, Spot 5 and Landsat 8.

How to use it ?

  1. Select an instrument
  2. Select an image (for RapidEye or Spot 5) or spectral band(s) (for Landsat 8)
  3. Select the metadata file. The button 'auto' tries to load automatically the metadata file.
  4. Click on the button 'get TOA Parameters' in order to find required parameters
  5. Select the output format : reflectance 32 bits [0-1], or milli-reflectance 16 bits [0-1000]
  6. Select the output directory (optional, by default the output file is written in the input file's directory)
  7. Click on the button convert 'DN to TOA' (be patient... The process could be quite long...)

Limitations

The plugin only works with GeoTIFF files.
For RapidEye and Spot images, the order of spectral bands must be as follows:

Instrument band 1 band 2 band 3 band 4 band 5
RapidEye blue green red rededge nir
Spot 5 nir red green swir -
